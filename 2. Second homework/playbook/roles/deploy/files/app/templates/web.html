About

This web version of the JSON request service is created just for fun and was not in the technical task. But I got some advantages from that experience: learned some Python, HTML, CSS and JS, lol.

Features:

  • You can send your JSON request via the top field;

  • You can choose HTTP method by the toggle switch. By default, it is GET request.

  • Emoji! To give an example, just put your cat (lol) in the colons like that: :cat: and you'll get an emojized 🐈!

Nothing more here.
